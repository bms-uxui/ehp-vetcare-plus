## 2. Notifications Screen Redesign

หน้าแจ้งเตือนออกแบบใหม่ตาม Figma

### Layout
- **StickyAppBar** (back + More) — fade-in title ตอน scroll
- **Hero** — title "แจ้งเตือน" + subtitle
- **Filter chips** (horizontal scroll): ทั้งหมด / ให้อาหาร / นัดหมาย / วัคซีน / ติดต่อ / การสั่งซื้อ
- **Reminder list** — Card ตาม type, มี unread dot + chip label

### iOS 26-style More menu
ปุ่ม More (ขวาบน) → กดแล้วปุ่ม **morph** เป็น dropdown card (ขยายจาก 44×44 → 240×150)
- ใช้ Reanimated + interpolate width/height/borderRadius พร้อม spring animation
- มี BlurView intensity 90 + tint ขาว 92% + shadow opacity 0.18
- 3 menu items: อ่านทั้งหมด / ตารางให้อาหาร / ตั้งค่าแจ้งเตือน
- ปุ่ม More จริงถูกซ่อน (`opacity: 0`) ตอน menu เปิด → ลอยทับเสมือนกัน

### New reminder types
เพิ่มใน [`src/data/reminders.ts`](../src/data/reminders.ts):
| Type | Icon | Color |
|---|---|---|
| `chat` | MessageCircle | ฟ้า `#4A8FD1` |
| `call` | Phone | เขียวมิ้นท์ `#3DA67A` |
| `order` | Package | ส้ม `#D17A4A` |

### Feeding reminder actions
แจ้งเตือนประเภท `feeding` มีปุ่ม **2 ปุ่ม** inline:
- "ให้อาหารแล้ว" (rose primary)
- "เลื่อน 15 นาที" (white + rose border)

### "อ่านทั้งหมด"
- ใช้ local state `readIds: Set<string>` ทับ `read: false` ของ mock data
- กดเมนูแล้วทุก reminder ที่ยังไม่อ่านจะ mark read ทันที (unread dot หาย)

## 9. Cross-cutting Polish

### Typography (ไทย tone marks)
แก้ปัญหาสระ/วรรณยุกต์ไทยถูกตัดด้านบน — ปรับ `lineHeight` ใน [`src/theme/typography.ts`](../src/theme/typography.ts):

| Variant | เดิม | ใหม่ |
|---|---|---|
| display | 34/42 | 34/**48** |
| h1 | 28/36 | 28/**40** |
| h2 | 22/30 | 22/**32** |

### Subtitle truncation
หลายหน้าเพิ่ม `numberOfLines={1}` + `adjustsFontSizeToFit` + `minimumFontScale={0.85}`:
- PetShop, VetHub, Cart, OrderTracking, ChatList, BookAppointment, BookTeleVet, Expenses

### Toggle UX
- Settings sheet ใช้ **native iOS Switch** (`<Switch>`) เพื่อให้ได้ visual + animation iOS 26 ที่แท้จริง — `trackColor.true: rose primary`, `ios_backgroundColor: '#E9E9EA'`
